Robofriends

Este pequeño proyecto se realizo con la finalidad de practicar React, lo que se puede hacer dentro de esta pagina es filtrar elementos, en este caso robots, los cuales tienen un nombre acompañado de información, lo interesante de esto es que la información es recibida al ingresar a la web y por medio del uso de promises la pagina espera a recibir todos los elementos en forma de JSON y poder cargarlos dentro de una variable para luego poder ser utilizada.

Background generator

En este caso la intención era incorporar JavaScript a una web y ver el impacto que este puede generar en una web que siendo solamente CSS y HTML seria prácticamente estática y no tendría interacción. Lo que se puede hacer en esta web es seleccionar dos colores para lograr un efecto de degradado con los colores seleccionados y en caso de obtener una combinación que sea agradable poder copiar el código que debería introducirse en la plantilla de “style.css” de una web para lograr el mismo resultado
